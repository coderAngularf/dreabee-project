API Documentation (For Developers)

Welcome to Dreabee’s API Documentation. Designed for developers, this guide provides all the information needed to integrate Dreabee’s powerful influencer marketing tools into your application. Our API gives you programmatic access to influencer data, audience insights, campaign management, and analytics. With Dreabee’s API, you can streamline data flow and enhance your marketing applications. Below, you’ll find setup instructions, endpoint details, and code examples to help you get started quickly and effectively.

Multi-Client Dashboard

Manage all client accounts in one place with Dreabee’s multi-client dashboard. Seamlessly switch between accounts, monitor campaigns, and track individual client metrics, all from a single, centralized platform.

The dashboard allows agencies to maintain an organized view of each client’s influencer activities, campaign progress, and performance metrics, making it easy to provide updates and insights for every brand you represent.
